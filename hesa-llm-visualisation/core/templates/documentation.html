Documentation

Overview

The HESA Data Analysis Dashboard is a powerful tool for analyzing and visualizing Higher Education Statistics Agency (HESA) data. It uses natural language processing to interpret user queries and generate meaningful visualizations.

Features

  • Natural language query processing
  • Interactive data visualizations
  • Multiple data export formats (CSV, Excel, PDF)
  • Summary statistics and insights
  • Historical data analysis

Usage Guide

Querying Data

Enter your query in natural language. For example:

  • "Show postgraduate enrollment trends from 2018 to 2022"
  • "Compare student satisfaction between Russell Group universities"
  • "Display international student numbers by year"

Downloading Data

Use the download buttons to export data in your preferred format:

  • CSV: Raw data format
  • Excel: Spreadsheet format with formatting
  • PDF: Report format with visualizations
